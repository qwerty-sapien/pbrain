# Notes

Generated Markdown is quarantined by default:

```text
Learning/Inbox/pb/
```

Useful commands:

```bash
pb notes inbox
pb notes organise --dry-run
pb notes organise --merge
pb notes organise --flatten
```

The goal is auditable Markdown first. `pb` should not silently scatter generated material through your vault.

Session evidence is different: `pb finish` creates a learner-facing evidence note for the active session. Reviews may also write daily or weekly notes when saved.
